\doxysubsection{Подробное описание}
Класс, представляющий звезду в симуляции. 

Звезда имеет координаты, скорость, массу, радиус, цвет и имя. Звёзды могут взаимодействовать друг с другом через гравитацию и столкновения. \begin{DoxyAuthor}{Автор}
kirbornu 
\end{DoxyAuthor}

\end{DoxyVersion}
